# 字典
## 特点
- 与集合类似，字典也是存储唯一值的数据结构，但是以**键值对**的形式来存储。
- ES6中有字典Map：Map翻译过来是映射的意思。
- 字典常用操作：键值对的增删改查。
```js
// 初始化
const m = new Map();
// 增
m.set('a', 'aa');
m.set('b', 'bb');
// 删
m.delete('a');
// 改
m.set('b', 'bbb');
// 全删
m.clear();
```
